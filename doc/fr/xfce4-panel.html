Introduction

Le panneau XFce 4 fait partie de l'environnement de bureau XFce 4 et comprend des lanceurs de programmes, des menus, une horloge, un changeur de bureau, et plus encore. La commande pour le lancer est xfce4-panel. Beaucoup d'éléments de ce panneau peuvent être configurés à partir d'une interface graphique.

Si vous êtiez habitué aux précédentes versions de XFce, vous trouverez beaucoup de nouvelles possibilités. Parmi les plus remarquables : la possibilité de mettre le panneau verticalement, les thèmes d'icônes, le nouveau pager graphique et la possibilité de configurer la position des boutons des menus popup. La manière dont les paramètres sont préservés est complétement modifiée, cela pourra prendre un peu de temps pour s'y habituer. Ce manuel devrait vous fournir toute l'information nécessaire pour adapter le panneau à vos préférences.

 

Démarrer

Quand vous lancez le panneau, vous voyez une barre au bas de votre écran, comme ceci :

Figure 1. Panneau XFce par défaut

L'illustration montre la configuration par défaut du panneau avec un menu ouvert et la souris au-dessus d'un bouton lanceur, faisant apparaître un message d'aide.

Il y a une poignée de déplacement à chaque extrémité du panneau. Vous pouvez les utiliser pour déplacer le panneau sur l'écran. Cliquer avec le bouton droit (ou Maj + clic gauche) sur l'une d'elles fait apparaître un menu popup... Allez-y, testez ;-)

Figure 2. Menu popup des poignées de déplacement

La zone située entre les poignées de déplacement contient les éléments du panneau. Les éléments disponibles sont décrits dans la section "Éléments du panneau". La plupart des éléments réalisent une action lors d'un clic. Le paramétrage des éléments se fait dans un menu accessible par un clic droit.

Un clic droit sur un item ouvre un menu popup à partir duquel vous pouvez ouvrir les propriétés de l'élément, supprimer l'élément ou ajouter un nouvel élément. Les éléments peuvent aussi être ajoutés à partir du menu popup des poignées de déplacement (voir au-dessus). Un sous-menu propose une liste des éléments disponibles dans laquelle choisir. Quand vous ajoutez un nouvel élément, le panneau ouvre le dialogue des propriétés, vous pouvez alors modifier les options de l'élément ainsi que sa position dans le panneau.

Figure 3. Eléments du menu popup

Les préférences globales du panneau peuvent être modifiées à partir du gestionnaire de paramètres du panneau XFce 4. Celui-ci peut être lancé, soit à partir du bouton lanceur du panneau, soit en choisissant 'Propriétés...' dans le menu popup des poignées de déplacement. Voir " Préférences du panneau" pour une explication des options disponibles.

 

Éléments du panneau

Le panneau Xfce 4 peut charger aussi bien des plugins externes que les plugins internes. Dans cette partie, nous décrirons seulement les éléments qui sont installés en même temps que le panneau.

La fenêtre de dialogue des propriétés d'un élément du panneau peut être ouverte avec un clic droit sur l'élément. Chaque type d'élément a son propre dialogue de configuration. Chaque fenêtre de dialogue possède au minimum ces options :

  • Changer la position d'un élément dans le panneau.
  • Supprimer cet élément.

Lanceur

L'élément le plus courant est le lanceur. Il vous permet de démarrer un programme à partir du panneau. La fenêtre de dialogue propose les options suivantes :

Commande
Vous pouvez entrer la commande à lancer ou le chemin à suivre dans le système de fichiers. Il existe une option pour lancer un programme dans un terminal et pour utiliser la notification de démarrage. Cette dernière option signifie que le gestionnaire de fenêtres montre un sablier pendant le chargement de l'application, à condition que celle-ci supporte cette fonctionnalité, ce qui est le cas de peu de programmes actuellement. Les programmes basés sur Gtk 2.2 sont censés la supporter.
Icône
Le panneau fournit un certain nombre d'icônes qui peuvent être utilisées en standard.Ces icônes peuvent aussi être harmonisées avec le thème d'icônes que vous avez choisi dans les préférences globales. Bien sûr, vous pouvez aussi sélectionner vos propres icônes en parcourant les répertoires grâce au bouton à coté de la zone de saisie, ou en déposant une image à partir du gestionnaire de fichier sur la zone de prévisualisation.
Bulle d'aide
Les éléments du panneau affichent cette aide si vous laissez la souris au-dessus d'eux.
Menu
Un lanceur peut contenir un menu. Quand cette option est choisie, un bouton avec une petite flêche apparaît à coté de l'élément. Reportez-vous à menus du panneau pour plus d'information. La position du bouton peut être modifiée dans les préférences générales.

Figure 4. Élément lanceur

Horloge XFce

Le style de l'horloge Xfce peut être modifié par certaines options. Il existe trois types d'horloge possibles : analogique, numérique et LED. Une option permet d'afficher ou non les secondes, et pour les modèles numérique et LED vous pouvez choisir un affichage sur 24 ou 12 heures (avec possibilité d'ajouter un mode AM/PM). L'horloge affiche en outre la date en bulle d'aide. Cliquer sur l'horloge n'apporte rien.

Figure 5. Élément horloge

Moniteur de courrier

Le moniteur de courrier surveille la boîte aux lettres locale de votre choix, à condition qu'elle soit au format mbox. L'icône de courrier s'accorde avec votre thème d'icônes. Vous pouvez choisir l'emplacement de votre boîte, la commande à lancer lorsque vous cliquez sur le bouton et l'intervalle entre chaque vérification. La commande a les mêmes options que celles des éléments lanceurs voir au-dessus).

Figure 6. Moniteur de courrier

Changeur de bureaux et changeur de bureaux graphique

Le panneau fournit deux éléments pour interagir avec les bureaux virtuels. Un changeur d'espace de travail de type CDE qui montre des boutons avec le nom des bureaux, et un changeur graphique qui affiche une vue miniature du bureau et vous permet même de déplacer une fenêtre vers un autre espace par un glisser-déposer. Le changeur graphique permet aussi de passer d'un espace à l'autre avec la molette de la souris quand le curseur est maintenu au-dessus.

Figure 7. Éléments sans option

Boutons système

Un élément "Boutons sytème" possède un ou deux boutons qui peuvent réaliser des actions spécifiques. Le bouton s'accorde avec votre thème d'icône. Il existe quatre boutons :

Verrouiller
Lance xfrun4 qui tente de verrouiller l'écran soit avec xscreensaver, soit, en cas d'échec, avec xlock.
Quitter
Ceci permet de fermer le panneau, et permet aussi de terminer votre session X (si vous utilisez startxfce4 ou avez placé le lancement du panneau à la fin de votre fichier de démarrage de X).
Information
Le bouton information amène le dialogue À propos de XFce 4.
Paramètres
Le bouton Paramètres ouvre le gestionnaire de paramètres de XFce, à partir duquel, vous accèdez aux réglages de tous les composants, ainsi qu'à certains réglages du système, tels le comportement de la souris et les paramètres de l'interface utilisateur.

Figure 8. Élément boutons système

Séparateur

Cet élément est simplement une ligne de séparation sur le panneau. Il n'a pas d'autre option que sa position.

Notez qu'il peut être difficile d'ouvrir son menu popup car vous avez trés peu de surface sur laquelle cliquer.

 

Menus du panneau

Les éléments lanceurs du panneau peuvent être associés avec un menu, lequel peut être ouvert en cliquant sur le bouton fléché à coté. La position de ce bouton est configurable dans les préférences globales. Vous pouvez aussi ouvrir le menu en cliquant sur le bouton lanceur et en prolongeant l'appui, le menu s'ouvre aprés un court délai.

Utilisez le choix 'Ajouter un lanceur' en première ligne du menu du lanceur, ou déposez un exécutable à partir du gestionnaire de fichier sur cette ligne, pour ajouter un nouveau choix au menu. Pour supprimer un élément du menu, utilisez le bouton 'supprimer' dans la fenêtre des propriétés de l'élément.

Une possibilité particulière des menus du panneau est qu'ils peuvent être détachés du panneau et rester dans leur propre fenêtre. Pour détacher un menu, cliquer sur la petite ligne en bas du menu (voir la copie d'écran du panneau par défaut).

Le dialogue des propriétés d'un élément du menu s'ouvre de suite quand vous faites un clic droit (ou Maj + clic gauche) sur celui-ci. Ce dialogue est trés semblable à celui de l'élément lanceur.

Figure 9. Elément de menu

 

Préférences du panneau

La fenêtre des propriétés du panneau XFce 4 peut être ouverte à partir du gestionnaire de paramètres de Xfce 4, lequel peut être ouvert par un lanceur du panneau ou par un clic droit sur les poignées de déplacement du panneau.

Figure 10. Paramètres du panneau

Apparence

Taille

Il y a quatre tailles disponibles : tout petit, petit, moyen, grand. La taille par défaut est petit.

Orientation

Une nouvelle fonctionnalité du panneau de XFce 4 est la possibilité d'être placé dans le sens vertical, aussi bien que dans le sens horizontal habituel.

Position des popup

Cette option modifie la position du bouton qui ouvre un menu du panneau. Il peut être placé au-dessus, en-dessous, à gauche ou à droite de l'élément lanceur.

Thème d'icônes

Une autre nouveauté de XFce 4 est la possibilité de choisir un thème d'icônes. Les icônes pour les lanceurs et les autres éléments du panneau suivent le thème d'icônes du panneau.

Position

Niveau

Le niveau du panneau conditionne sa position par rapport aux autres fenêtres. Le panneau peut être en permanence au-dessus, caché par les autres fenêtres, ou agir comme les autres fenêtres. Par défaut, le panneau se situe au-dessus.

Centrage

Il est trés facile de déplacer le panneau sur l'écran. Le dialogue de paramétrage vous permet de centrer le panneau sur l'un des cotés de l'écran. Notez que le panneau 'se colle' aux coins de l'écran tout comme au milieu d'un coté de celui-ci si vous l'amenez suffisamment prés d'une de ces positions.

Comportement

Si vous cochez le choix 'Masquer automatiquement', le panneau se transforme en une fine barre. Déplacer la souris au-dessus de celle-ci découvre le panneau. Ceci fonctionne mieux quand le panneau est sur un bord de l'écran (la cible est plus facile à toucher).
